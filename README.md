## About
Neon Maze Game is a colorful browser maze challenge with:
- Multiple difficulties: Easy, Hard, Impossible
- Whole-maze coin scatter
- Enemy gates that require points to pass
- Theme switching with animated neon styles
- Keyboard and touch/swipe support

## Controls
- Move: Arrow keys or WASD
- Mobile: Swipe on the maze
- New run: New Maze button
- Theme switch: Themes button

## Gameplay Notes
- Easy: Collect and explore freely.
- Hard: No immediate backtracking.
- Impossible: No immediate backtracking plus step limits.
- Bigger grids increase challenge and scale visuals for readability.

## Project Structure
- index.html: App layout and UI structure
- style.css: Theme system, animations, responsive design
- game.js: Maze generation, movement, enemies, scoring, game flow

## Quick Start
1. Clone or download the project.
2. Open index.html in any modern browser.
3. Pick a difficulty and start playing.
